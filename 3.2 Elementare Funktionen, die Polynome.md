#Def/3/9 
Das #Horner-Schema ist eine tabellarische Form der Polynomauswertung. Es basiert auf der Darstellung
	$P_{n}(x) = a_{0} + a_{1} x + \dots + a_{n} x^n$
	$= a_{0} + x \cdot (a_{1} + x \cdot (a_{2} + x \cdot (\dots + x\cdot(a_{n-1} + x \cdot a_{n}) \dots)))$
eines gegebenen Polynoms $P_{n}$. Für ein festes gegebenes $x = x_{0}$ werden die Klammerausdrücke von innen nach außen ausgewertet. D.h. wir berechnen nacheinander:
1. $b_{n} = a_{n}$
2. $b_{n-1} = a_{n - 1} + x_{0} \cdot b_{n}$
3. $b_{n-2} = a_{n-2} + x_{0} \cdot b_{n-1}$
	$\vdots$
4. n-ter Schritt) $b_{1} = a_{1} + x_{0} \cdot b_{2}$
5. Schritt $n+1$: $P_{n}(x_{0}) = b_{0} = a_{0} + x_{0} \cdot b_{1}$
"Multipliziere $x_{0}$ mit dem vorherigen Ergebnis und addiere den aktuellen Koeffizienten"
=> Schema ist eine Tabelle mit **allen** Koeffizienten absteigend aufgeschrieben in der ersten Zeile, in der zweiten Zeile $z_{0} \cdot b_{k}$ und das Ergebnis $b_{k-1}$ in der dritten Zeile
		$p(x) = x^{3} + x -3, \quad x_{0} = 1$
		

|             | 1   | 0   | 1   | -3          |
| ----------- | --- | --- | --- | ----------- |
| $x_{0} = 1$ |     | 1   | 1   | 2           |
|             | 1   | 1   | 2   | -1 $= p(1)$ |

#Bem/3/11 Es gilt:
 $P_{n}(x) = a_{n}x^n + \dots + a_{1} x + a_{0}$
	 $= (b_{n}x^{n-1} + \dots + b_{2} x + b_{1})(x - x_{0}) + b_{0}$
	 $\iff P_{n}(x) = P_{n-1}(x) \cdot (x - x_{0}) + b_{0}$
	 $\iff \frac{P_{n}(x)}{x - x_{0}} = P_{n-1}(x) + \frac{b_{0}}{x - x_{0}}$

Falls $b_{0} = 0$ gilt und $x_{0}$ somit Nullstelle von $P_{n}$ ist, so führt das Horner-schema gleichzeitig die #Polynomdivision mit dem #Linearfaktor $x - x_{0}$ aus.
Die Koeffizienten des Polynoms $P_{n-1}$ stehen in der letzten Zeile.

#Def/3/13
Die #Newton-Darstellung ist eine Darstellung von Polynomen, speziell für die Interpolation durch gegebene #Stützstellen $\{ x_{0}, x_{1}, \dots, x_{n} \}$ mit den #Basispolynome
	$N_{0}(x) := 1$
	$N_{1}(x) := (x - x_{0}) \cdot N_{0}(x)$               $\implies N_{1}(x) = x - x_{0}$ 
	$N_{2}(x) := (x - x_{1}) \cdot N_{1}(x)$              $\implies N_{2}(x) = (x - x_{1})(x - x_{0})$
	$N_{n}(x) := (x - x_{n - 1}) N_{n-1}(x)$         $\implies N_{n}(x) = \prod_{k=0}^{n=1}(x- x_{k})$
	
Ein polynom $n$-ten Grades besitzt dann für gegebene Koeffizienten $c_{0}, \dots, c_{n}$ die Darstellung
	$P_{n}(x) = c_{n}N_{n}(x) + c_{n-1} N_{n-1}(x) + \dots + c_{0} N_{0}(x)$
	
Die Idee der Interpolation ist es nun, für ein geeignetes $n$ die #Interpolationsbedingungen 
	$y_{i} = P_{n}(x_{i}), \quad i \in \{ 0, 1, \dots, n \}$
zu fordern. Diese Aufgabe besitzt für paarweise verschiedene Stützstellen eine eindeutige Lösung, das #interpolationspolynom
