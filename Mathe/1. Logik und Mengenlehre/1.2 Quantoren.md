Um Aussagen über eine Klasse von Objekten treffen zu können, benötigen wir neue Symbole

Def 1.15)
Der #Allquantor $\forall$ auch $\bigwedge$, leitet einen Ausdruck mit "für alle ..." ein
Der #Existenzquantor $\exists$l, auch $\bigvee$ leitet einen Ausdruck mit "es existiert ein ..." ein
Speziell $\exists!$ leitet einen Ausdruck mit "es existiert genau ein ..." ein

Bsp 1.16)
- Es gibt eine natürliche Zahl, die gerade und größer als 5 ist
$\exists \quad n \in \mathbb{N} \quad: \quad 2\mid n \ \land \ n<5$

Aufgabe: "Es gibt eine gerade Primzahl"
$\exists \quad n\in\mathbb{N}\quad :\quad 2\mid n \ \land n \in \mathbb{P}$
oder 
$\exists \quad n\in\mathbb{N}\quad :\quad 2\mid n \ \land (\exists m \in \mathbb{N} : m \mid n \land m \neq 1 \land m \neq n)$
-> Satz ist wahr, n=2 ist einzige -> "$\exists!$"

Für alle reelen Zahlen ist deren Quadrat nicht negativ
$\forall x \in \mathbb{R} \quad : \quad x^2 \geq 0$

Für alle epsillon größer als null gibt es eine natürliche Zahl N, mit 1/n ist kleiner als e
$\forall \epsilon \in \mathbb{R} \quad \exists n \in \mathbb{N} \quad : \quad \epsilon > 0 \ \land \ \frac{1}{n} < \epsilon$
-> Wahr, da für jedes feste e ein n> 1/e gewählt werden kann

$\exists n \in \mathbb{N} \quad \forall \epsilon \in \mathbb{R} \quad : \quad \epsilon > 0 \ \land \ \frac{1}{n} < \epsilon$
-> "Es existiert eine natürliche Zahl n, so dass für alle e größer Null gilt, dass 1/n kleiner als e ist"
-> Falsch, es gibt keine natürliche Zahl die so groß ist, dass 1/n kleiner als jedes e>0 ist


Bem 1.17)
Die Reihenfolge der Quantoren in einem logischen Ausdruck ist wichtig da z.B bei unterschiedlichen Quantoren die hinteren von den vorderen abhängen
Die Negation von Quantoren erfolgt gemäß
- $\overline{\forall x : p(x)} = \exists x : \overline{(p(x))}$
  "nicht für alle x gilt p(x)" = "es existiert ein x mit nicht p(x)"
- $\overline{\exists x : p(x)} = \forall x : \overline{(p(x))}$
  "es existiert kein x mit p(x)" = "für alle x gilt nicht p(x)"
