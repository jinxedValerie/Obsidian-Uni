### Def 1.9)
Die #Implikation (wenn dann Verknüpfung) ist genau dann falsch, wenn der erste Ausdruck wahr und der zweite falsch ist 
Symbol: $p\implies q$, gesprochenen: "p impliziert q" oder "aus p folgt q"
Wahrheitstafel:

| $pq$ | $p\implies q$ |
| ---- | ------------- |
| ww   | w             |
| wf   | f             |
| fw   | w             |
| ff   | w             |

### Def 1.11)
Die #Äquivalenz (genau dann wenn Verknüpfung) ist genau dann wahr, wenn beide Ausdrücke den gleichen Wahrheitswert haben
Symbol $p \iff q$, gesprochen "p genau dann wenn q"
Tabelle:

| $pq$ | $p \iff q$ |
| ---- | ---------- |
| ww   | w          |
| wf   | f          |
| fw   | f          |
| ff   | w          |

### Bem 1.13)
Bei der Verknüpfung logischer Ausdrücke ist auf die #Priorität der Operationen zu achten.
Von stark zu schwach hier:
1. $\neg$
2. $\land$, $\lor$
3. $\implies$
4. $\iff$

x,y logische / numerische Variablen

| logische Operationen | boolesche Operation |
| -------------------- | ------------------- |
